# Welcome to the Demo Page

This is a demo page created using MDX for Docusaurus. MDX allows you to write JSX within your Markdown files, making it easy to create interactive and dynamic content.

## Features

- **Easy to use**: Write Markdown and JSX in the same file.
- **Interactive**: Add React components to your documentation.
- **Customizable**: Style your components with CSS.
